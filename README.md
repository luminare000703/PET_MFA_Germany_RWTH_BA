# PET_MFA_Germany_RWTH_BA
Materialflussanalyse (MFA) von PET-Verpackungen in Deutschland mit Szenarioanalyse von Recyclingpfaden und politischen Zielvorgaben

Modellierung mit ODYM auf Basis von Excel-Parametern. Der Code ist für die Durchführung von Szenarioanalysen ausgelegt, wobei sich die Implementierung je nach Szenario unterscheidet. In einzelnen Fällen wurden Parameter fest im Code hinterlegt (Hardcoding), weshalb bei der Anwendung Vorsicht geboten ist. Zudem sollten Unsicherheitsanalysen (Monte-Carlo-Simulationen) für alle Szenarien außer dem Basisszenario nicht berücksichtigt werden.
